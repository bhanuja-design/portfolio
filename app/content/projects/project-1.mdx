# Project One

A case study exploring how thoughtful design can simplify complex workflows.

## Problem

Users struggled to complete key tasks because the existing interface buried important actions behind multiple layers of navigation. Completion rates were low and support tickets were climbing.

## Approach

I mapped the end-to-end user journey, identified friction points, and prototyped a streamlined layout that surfaced the most common actions upfront. Iterative testing with five users shaped the final information architecture.

## Outcome

Task completion improved and the redesigned flow reduced average time-on-task. The project established patterns that were later adopted across other product areas.
